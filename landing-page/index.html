Perguntas Frequentes

Preciso de um servidor dedicado?

Não necessariamente. O Nimbus roda em qualquer VPS com 2GB+ de RAM. Recomendamos DigitalOcean, Hetzner ou AWS EC2 para produção.

Funciona com qualquer framework?

Sim! Suporta Next.js, React, Vue, Nuxt, Express, Fastify, Python, Go, Dart e mais. O Nimbus detecta automaticamente o framework.

Como funciona o backup?

Banco de dados PostgreSQL com backup automático diário. Você também pode exportar manualmente a qualquer momento.

Posso usar domínio próprio?

Sim! No plano Pro e acima, você pode configurar domínios customizados com HTTPS automático via Let's Encrypt.

Pronto para começar?

Faça deploy do seu primeiro projeto em menos de 5 minutos. Código aberto, sem lock-in.
