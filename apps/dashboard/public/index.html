Features

Tudo que você precisa

Da entrega contínua ao monitoramento. StackRun empacota o workflow completo de deploy em uma plataforma self-hosted.

🚀

Git Push to Deploy

git push e seu app está no ar em segundos. Sem configuração manual de CI/CD.

📦

Docker + Compose

Suporte nativo a Dockerfile e docker-compose.yml. Suas stacks rodam exatamente como você define.

🔒

SSL Automático

Certificados TLS provisionados automaticamente para todos os domínios via Caddy.

🗄️

Bancos Gerenciados

PostgreSQL provisionado com 1 clique. Backups automáticos e conexão segura.

🌐

Domínios Customizados

Seu domínio, sua marca. Conecte quantos quiser sem custo adicional e sem lock-in.

📊

Métricas em Tempo Real

CPU, RAM, disco, uptime e logs ao vivo. Tudo em um dashboard unificado.

🤖

CI/CD Nativo

GitHub Actions, webhooks configuráveis e preview deploys para cada pull request.

🔐

RBAC

Roles de admin, developer e viewer por projeto. Controle granular de acesso ao seu time.
